# Design System

## Direction

Professional mobile learning product with restrained color, strong hierarchy, generous spacing, and familiar Android interaction patterns.

## Color

- Primary: deep navy blue for primary actions and current navigation.
- Background: cool near-white.
- Surface: white and light blue-gray.
- Success: restrained green.
- Error: restrained red-orange.
- Text: near-black primary and blue-gray secondary.

## Typography

Use the Android system sans family. Titles rely on weight and spacing rather than decorative fonts. Body copy stays at 16sp or above for question content.

## Components

- Primary buttons use one consistent rounded rectangle shape.
- Cards are reserved for selectable learning objects or grouped status, not every section.
- Bottom navigation contains four stable destinations: 首页、题库、错题、我的.
- Answer options use a full-row selection target with an explicit radio control.

## Layout

Use a 20dp horizontal page gutter, 8dp spacing unit, and 16-20dp corner radius. Keep primary actions near the lower thumb zone during quizzes.
