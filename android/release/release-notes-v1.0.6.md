## 题域引擎 1.0.6

### 体验优化

- 「我的」页精简：学习数据与「应用设置」分离，设置区从 9 条收敛为 4 组入口
- 新增学习报告子页：趋势、薄弱题库、题型正确率、最近测验
- 期末模拟卷合并为单一入口，支持选卷 A~G 与随机抽卷
- 答题卡改为 Bottom Sheet，格子更大、更易点选
- 答题退出增加确认，防止误触丢失进度
- 返回键层级修正：结果页回到来源页，设置子页逐级返回
- 结果页默认只看「需要巩固」，有错时 AI 学情默认折叠
- AI 共享模式冷却改为友好提示与倒计时，不再显示为报错
- 结果页解析卡片支持收藏/取消收藏
- 启动时检查更新（有网时），可一键前往更新页
- 「我的」页支持下拉刷新同步题库
- 无网络时顶部离线横幅，AI/同步操作给出明确提示
- 页面切换增加淡入滑动过渡
- 跟随系统深色模式（设置、报告、题库等核心页面）

### 数据与质量

- 掌握度统计改为按题目最近一次作答计算，更准确反映当前水平
- 设置页新增「隐私说明」入口，跳转公开隐私政策页
- 新增 GitHub Actions 质量检查（Detekt + Lint + 单元测试）
- 新增学习备份与版本比较单元测试

### 说明

- 本版本不包含每日学习提醒
- 本版本不包含随机组卷（除期末卷随机抽卷外）
