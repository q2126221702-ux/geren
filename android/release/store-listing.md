# 题域引擎商店资料

## 应用名称

题域引擎

## 简短说明

工业网络与英语题库练习，支持错题复习、学习统计和在线题库同步。

## 完整说明

题域引擎是一款专注于工业网络与英语备考的本地学习工具。

主要功能：

- 多套离线题库，安装后无需联网即可练习
- 支持单选、多选、判断、填空和问答题
- 自动记录错题，并按照掌握进度安排复习
- 展示近 7 天趋势、题型正确率和薄弱题库
- 优先推荐到期错题和需要加强的题库
- 可从 GitHub 主动同步最新公开题库，或在「我的」页下拉刷新
- 可在设置中检查并安装开发者签名的新版本；启动时也会提示有新版本
- 可导出和导入学习记录与错题进度，更换设备也能恢复
- 支持收藏题目、按题库和题型筛选，并在结果页解析卡片上收藏
- 应用设置分组管理：备份、AI、更新、关于；学习报告独立入口
- 期末模拟卷支持选卷与随机抽卷
- 跟随系统深色模式
- 学习记录保存在本机，无账号、无广告、无行为追踪

网络仅用于用户主动发起的题库同步。即使同步失败，现有离线题库和学习记录仍可继续使用。
