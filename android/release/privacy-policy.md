# 题域引擎隐私说明

更新日期：2026 年 7 月 12 日

题域引擎是一款本地题库与错题复习应用。我们重视你的隐私，并按照以下方式处理数据。

## 数据处理

- 答题记录、错题进度、学习统计和题库同步时间保存在你的设备上。
- 应用不要求注册账号，不收集姓名、手机号、通讯录、位置、相册、麦克风或广告标识符。
- 应用不包含广告、行为追踪或第三方统计 SDK。

## 网络访问

应用仅在你主动点击「立即同步题库」或下拉刷新时访问 GitHub 下载公开题库文件。该过程不上传答题记录或其他个人数据。网络请求可能由 GitHub 按其服务规则记录常规连接信息，例如 IP 地址和访问时间。

应用在有网络时可能于启动后检查 GitHub Releases 是否有新版本；在你主动进入「检查更新」并确认下载后，从 GitHub Releases 获取由开发者签名的新版本 APK，并调用 Android 系统安装界面。应用会拒绝安装签名与当前版本不一致的 APK；不会静默安装软件。

## AI 功能

当你主动使用 AI 解析、评分或学情分析时，应用会将当前题目、你的作答和必要的上下文发送至你所选择或站点默认的 AI 服务商，以生成结果。该过程由你主动触发，不会在后台自动上传学习记录。

自带 API Key 使用 Android 系统密钥加密，仅保存在本机，不会进入学习数据备份，也不会上传到题域引擎服务器。不同 AI 服务商可能按其各自隐私政策处理请求内容，请在使用前了解对应服务商规则。

## 备份与删除

Android 系统可根据你的系统备份设置备份学习记录、错题进度和同步时间。已下载题库不会进入系统备份。卸载应用或在系统设置中清除应用数据，可删除设备上的全部本地数据。

当你主动使用“导出学习数据”时，应用会将学习记录、错题进度和收藏题目 ID 写入你通过 Android 系统文件选择器指定的 JSON 文件。应用不会自动上传该文件；你后续通过微信、网盘或其他方式分享该备份时，应自行保管并控制接收范围。导入前应用会显示记录数量并要求确认覆盖。

## 儿童隐私

应用不主动收集任何用户的个人信息，也不针对儿童投放广告或进行画像。

## 变更与联系

如隐私处理方式发生变化，本说明会同步更新。问题或建议可通过应用题库项目的 GitHub 仓库反馈。
